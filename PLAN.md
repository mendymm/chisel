# Chisel — Coding Agent

A terminal-based coding agent written in 100% Rust.

---

## MVP Plan

---

## Feature Ideas (Post-MVP)

- [ ] Multi-turn conversation with persistent history
- [ ] Context window management / summarization
- [ ] Search/grep tool
- [ ] Glob / file search tool
- [ ] Web fetch tool
- [ ] Configurable system prompt
- [ ] Config file (`~/.chisel/config.toml`)
- [ ] Support multiple LLM providers
- [ ] Token usage tracking and display
- [ ] Syntax-highlighted output
- [ ] Undo / rollback file changes

## Other features

- [ ] a `/tmp` and `--tmpdir` command that will create a tmp dir, and spawn the agent in that tmp dir 
- [ ] a `/btw` command to ask the agent questions, isolated from the general convo context
