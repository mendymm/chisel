# Agent Instructions

This project is a coding agent written in 100% Rust.

## General

- Keep it simple. Prefer clear, idiomatic Rust over clever abstractions.
- Do not add deps yourself, ask the user to add the deps.
- You are forbidden from executing the code/tests, do not run `cargo run/cargo test`
- You are allowed to check if the code compiles with `cargo check` or `cargo build`, prefer `cargo check`
- You can use cargo doc to generate html documentation of crates
